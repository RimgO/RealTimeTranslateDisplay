# 字幕自動サイズ調整機能

## 実装日
2026-01-31

## 概要
字幕表示エリアの高さを固定し、表示文字数に応じて自動的にフォントサイズを調整する機能を実装しました。

## 変更ファイル
- `/src/main.ts` - 自動フォントサイズ調整ロジックの実装
- `/src/style.css` - 字幕エリアの固定高さとスタイル調整

## 機能詳細

### 自動サイズ調整の仕組み
1. **基準フォントサイズ**:
   - 原文（original）: 1.5rem
   - 翻訳文（translated）: 2.5rem

2. **最小フォントサイズ**: 0.8rem

3. **文字数しきい値**:
   - 原文: 100文字
   - 翻訳文: 80文字

4. **調整アルゴリズム**:
   - 文字数がしきい値以下: 基準サイズを使用
   - 文字数がしきい値超過: `√(しきい値 / 文字数)` の比率でサイズを縮小
   - 最小サイズ以下にはならない

### CSS変更
- 字幕エリアの高さを `30vh`（最小150px、最大40vh）に固定
- `word-wrap` と `overflow-wrap` で長い単語も適切に折り返し
- `transition: font-size 0.3s ease` でスムーズなサイズ変更

### 適用箇所
以下のすべてのテキスト更新時に自動調整が適用されます:
- 音声認識結果の表示
- MLXサーバーからの認識・翻訳結果
- Google Translate APIの翻訳結果
- Ollama翻訳結果

## 使用方法
特別な設定は不要です。字幕が表示されると自動的にサイズが調整されます。

手動でのフォントサイズ調整（スライダー）も引き続き使用可能で、その場合は手動設定が基準値として使用されます。
